    6 Informasi
    Parameter input dan output

    Parameter Input

    • Q = debit rencana yang dimasukkan user, satuan m³/s.
    • Jenis sedimen = pilihan material dominan: lanau lunak, lanau sedang, clay, atau lanau padat.
    • D₅₀ = diameter butiran median. Nilai ini dipakai untuk membaca karakteristik kecepatan pengendapan.
    • C₀ = konsentrasi sedimen masuk/influen, satuan mg/L.
    • N = jumlah kompartemen paralel.
    • h = kedalaman air rencana, satuan m.
    • v = kecepatan aliran rencana, satuan m/s.
    • w = kecepatan endap efektif sedimen, input dalam mm/s lalu dikonversi ke m/s.

    Parameter Output

    • As = luas permukaan kolam minimum, dihitung dari Q/w.
    • A = luas penampang basah, dihitung dari Q/v.
    • Btotal = lebar total kantong lumpur, dihitung dari Q/(v×h).
    • Bkomp = lebar per kompartemen, dihitung dari Btotal/N.
    • L = panjang kantong lumpur, dihitung dari As/Btotal.
    • L/B = rasio dimensi terhadap lebar total.
    • T = waktu retensi, dihitung dari volume air aktif/Q.
    • η = efisiensi pengendapan.
